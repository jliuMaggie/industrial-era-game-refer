# 家族穿越：工业革命纪元 — 游戏设计研究资料

## 游戏概述
模拟经营+肉鸽卡牌+历史穿越游戏。玩家带领家族穿越四次工业革命（1760s→2020s），通过投资决策成为万亿豪门。

## 核心数据规模
- 27种投资物 × 7级升级 = 189个投资等级
- 12种时代危机（每时代3种，随机触发1-3种）
- 12种时代机遇（每时代3类，随机触发1-3种）
- 5个AI竞争家族
- 40个联姻对象（每时代10个）
- 37个继承人天赋
- 42个成就

## 游戏界面/屏幕（相当于"页面"）

### 1. 开始界面 (Start Screen)
- 游戏标题、史诗背景图
- 开始新游戏 / 继续游戏 按钮
- 家族名称输入
- 背景音乐开关

### 2. 主游戏界面 (Main Game HUD)
- **顶部状态栏**: 年份、时代标签、资金、声望、排名
- **左侧边栏**: 当前投资列表（可升级）
- **中央主区域**: 
  - 三选一投资决策卡片
  - 投资按钮（全仓/单笔）
- **右侧边栏**: AI家族排名、当前家族信息
- **底部**: 当前继承人信息、快捷操作

### 3. 投资详情弹窗 (Investment Modal)
- 投资物卡片大图
- 7级升级路线可视化（肉鸽技能树风格）
- 各级数值对比
- 全仓/单笔投资按钮
- 暴击率预览

### 4. 事件弹窗 (Event Modal)
- 危机降临：黑色/红色主题，历史文案，效果展示
- 机遇降临：金色主题，历史文案，效果展示
- 传说机会：彩虹金光特效

### 5. 暴击特效层 (Crit Overlay)
- 全屏半透明覆盖
- 中央大字"暴击！"+倍率
- 数字跳动动画
- 彩色纸屑粒子效果
- 连击计数器

### 6. 时代跨越动画 (Era Transition)
- 全屏黑色过渡
- 史诗历史文案逐行显示
- 新时代解锁内容展示
- 背景音乐切换

### 7. 联姻界面 (Marriage Screen)
- 抽卡风格界面
- 候选人卡片翻转/抽拉动画
- 品质光效（普通白/稀有蓝/史诗紫/传说金）
- 刷新按钮（消耗金币）

### 8. 继承人界面 (Heir Screen)
- 天赋卡牌展示
- 培养方向选择（6个方向）
- 投入金额滑块
- 传承预览

### 9. 排名界面 (Ranking Screen)
- 家族财富排行榜
- 趋势图表
- AI家族行动日志

### 10. 成就界面 (Achievements)
- 成就网格展示
- 已解锁/未解锁状态
- 进度条

### 11. 游戏结束界面 (Game Over)
- 胜利：万亿帝国庆典画面
- 失败：家族破产/绝嗣画面
- 完整家族史时间线

## 视觉风格要求
- **整体风格**: 复古奢华 + 工业革命元素 + 现代卡牌游戏UI
- **色调**: 
  - 第一次工业革命：暖棕褐色（蒸汽、煤炭、铜）
  - 第二次工业革命：钢铁蓝灰色
  - 第三次工业革命：电子绿色
  - 第四次工业革命：科技紫色
- **卡牌设计**: 类似小丑牌/杀戮尖塔的卡牌风格，品质带光效边框
- **特效**: Canvas粒子系统（暴击纸屑、传说金光）
- **字体**: 标题用衬线体（历史感），正文用无衬线体

## 技术约束
- React 19 + TypeScript + Tailwind CSS v3 + Vite
- 单页应用（HashRouter切换不同"屏幕"）
- 使用shadcn/ui组件
- 需要Canvas 2D做粒子特效
- 无需后端（纯前端游戏）

## 核心爽感设计
1. 高频暴击反馈（12%-50%暴击率）
2. 连击倍率递增
3. 传说投资超高回报（12-30倍）
4. 投资升级可视化（技能树风格）
5. 时代跨越仪式感
6. 排名竞争紧迫感
