## 一、游戏概述

### 1.1 核心定位
- 模拟经营 + 家族养成 + 历史穿越
- 核心玩法：带领家族穿越四次工业革命，通过投资决策成为万亿豪门
- 差异化卖点：借鉴小丑牌彩票逻辑的爽感设计 + AI家族排名竞争

### 1.2 核心卖点
- 跨越300年历史：1760s → 2020s
- 复利的力量：祖辈财富如何通过时间指数级增长
- 时代特色投资：每个时代独特的投资机会和风险
- 家族传承机制：选择继承人、培养家族天赋、延续商业帝国
- 小丑牌式爽感：暴击、连击、传说投资机会
- AI家族竞争：与其他家族同台竞争排名

## 二、爽感机制设计（借鉴小丑牌）

### 2.1 暴击系统
```
┌────────────────────────────────────────────────────────────────┐
│                        暴击系统                                  │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│  【基础暴击】                                                   │
│  • 每次投资默认5%暴击率                                        │
│  • 暴击时收益 ×3~×10（随机）                                   │
│  • 暴击触发金光特效 + 数字跳动动画                              │
│                                                                │
│  【连击机制】                                                   │
│  • 连续暴击触发"连击"                                          │
│  • 第2次暴击：×5                                               │
│  • 第3次暴击：×10                                              │
│  • 第N次暴击：×(N+2)                                           │
│  • 连击中断则重置                                               │
│                                                                │
│  【暴击蓄力】                                                   │
│  • 未暴击时积累"暴击能量"                                       │
│  • 能量满时下一次投资必暴击                                     │
│  • 不同投资类型蓄力速度不同                                     │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

## 六、时代设计

### 6.1 四次工业革命时代设定

#### 第一次工业革命（1760-1840）：蒸汽与纺织
```
时代特征：
• 蒸汽机发明，机械化生产开始
• 纺织业崛起，工厂制度诞生
• 铁路建设改变交通
• 城市化进程加速

进入条件（门槛）：
• 资产 ≥ 10万
• 声望 ≥ 200
• 家族成员 ≥ 1人

传说机会：
• 瓦特蒸汽机专利（传说级）
• 铁路特许经营权（史诗级）
• 纺织厂规模化（稀有级）
```

#### 第二次工业革命（1870-1914）：电气与钢铁
```
时代特征：
• 电力广泛应用，内燃机发明
• 钢铁和石油工业大发展
• 汽车、飞机相继发明
• 垄断资本主义形成

进入条件（门槛）：
• 从第一次工业革命时代通过
• 资产 ≥ 50万
• 声望 ≥ 500
• 持有至少2个工业类投资项目

传说机会：
• 石油开采权（传说级）
• 汽车生产线（史诗级）
• 电气公司（稀有级）
```

#### 第三次工业革命（1945-2000）：电子与信息
```
时代特征：
• 计算机和互联网诞生
• 航空航天技术突破
• 电视和传媒产业爆发
• 全球化加速

进入条件（门槛）：
• 从第二次工业革命时代通过
• 资产 ≥ 500万
• 声望 ≥ 2000
• 持有科技类投资或完成科技成就

传说机会：
• 半导体公司收购（传说级）
• 航天投资机会（史诗级）
• 电视传媒帝国（稀有级）
```

#### 第四次工业革命（2000-至今）：智能与数字
```
时代特征：
• 人工智能和大数据
• 移动互联网普及
• 新能源革命
• 生物科技突破

进入条件（门槛）：
• 从第三次工业革命时代通过
• 资产 ≥ 5000万
• 声望 ≥ 8000

│  • 保持玩家正向情绪，不因失败弃游                                           │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
