# 家族穿越：工业革命纪元 - 游戏开发计划

## 项目概述
基于《家族穿越：工业革命纪元 V2 小丑牌增强版》策划案，开发一款完整的可玩Web游戏。
技术栈：React 18 + TypeScript + Tailwind CSS + HTML5 Canvas（特效）

## 核心需求
1. 完整还原策划案所有功能、文案、数值
2. 肉鸽技能升级路线：每种投资物5+等级，初级完成后解锁高级选项
3. 每个时代3种危机，随机触发1-3种
4. 每个时代3类机遇，随机触发
5. 投资支持"全仓买入"和"单笔买入"
6. 完整的暴击系统、连击系统、传说机会系统
7. AI家族竞争、联姻抽卡、继承人天赋
8. 完整的成就系统

## Stage 1 — 游戏数据与架构设计
- 整合策划案所有文案、数值到结构化数据文件
- 设计投资物升级路线（每类投资5-7个等级）
- 设计时代危机库（每个时代3种）
- 设计时代机遇库（每个时代3类）
- 定义游戏核心状态机
- 输出：完整的游戏数据JSON + 类型定义

## Stage 2 — 美术资产生成
- 生成4个时代的投资卡牌美术（普通/稀有/史诗/传说品质）
- 生成危机/机遇事件背景图
- 生成家族头像、继承人头像
- 生成UI元素、按钮、图标
- 输出：PNG/JPG资产包

## Stage 3 — 前端核心开发（多Agent并行）
### 3A - 游戏引擎与状态管理
- GameState 管理器（资金、声望、时代、年份、排名）
- 投资计算引擎（收益、暴击、连击、失败判定）
- 时代推进系统（门槛判定、跨越动画）
- AI家族模拟系统

### 3B - UI组件开发
- 主界面布局（顶部状态栏、侧边栏、主区域）
- 投资卡牌组件（品质边框、悬浮效果、暴击动画）
- 三选一决策界面
- 危机/机遇弹窗
- 联姻抽卡界面
- 继承人界面
- 排名面板

### 3C - 特效与动画
- 暴击特效（Canvas粒子+数字跳动）
- 连击特效
- 传说卡金光特效
- 时代跨越转场动画
- 成就解锁动画
- 纸屑飘落效果

## Stage 4 — 集成与部署
- 所有模块集成
- 完整游戏流程测试
- 部署到生产环境
- 输出：可访问URL

## 技能加载
- Stage 1: 自主设计
- Stage 2: 自主设计（generate_image）
- Stage 3: vibecoding-webapp-swarm
- Stage 4: deploy_website
